PhotoBooth

PhotoBooth is a photography website built to showcase real moments, real energy, and real vibes.  

This project is a simple, clean photography portfolio created using HTML and CSS, focused on layout, styling, and user experience.

Features

- Home Page – Introduces the PhotoBooth vibe and aesthetic  
- Memories Page – A grid gallery showcasing captured moments  
- Portfolio Page – Highlights selected photography work  
- Top Navigation Bar – Easy navigation between pages  
- Modern Styling – Hover effects, spacing, and responsive layouts  

 Built With

- HTML5  
- CSS3
